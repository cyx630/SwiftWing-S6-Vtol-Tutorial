# 飞控教程

## 飞控系统介绍

飞控是飞行器的核心控制单元，负责姿态稳定、导航和任务执行。

## 飞控校准步骤

1. 连接飞控到地面站
2. 进行加速度计校准
3. 进行罗盘校准
4. 检查参数设置
