# 起飞前地面测试

## 地面测试流程

起飞前必须完成以下地面测试，确保飞行器状态正常。

## 测试项目

1. 遥控器信号测试
2. 舵面行程检查
3. 电机运转测试
4. GPS 搜星状况
5. 飞行模式切换测试
