# 准备通信链路

* 硬件准备为每次飞行都需要进行
* 软件准备为第一次，按照教程配置软件连接。后续飞行，按需调整对应配置即可
* 高频头出厂为无线蓝牙连接，有线连接如下：

![1783749736557](image/准备通讯链路（有线连接）/1783749736557.png)

## 硬件准备

1. MLRS高频头
2. 遥控器、接收机
3. VtolS3无人机主体
4. 无人机电池
5. 电脑

### 硬件连接

1. 将转接座对准遥控器高频头卡槽，对准排针，确保安装没有虚位

![1776931255179](image/准备通信链路/1776931255179.png)

2. 根据高频头上标识，将天线连接到高频头上

![1776931158160](image/准备通信链路/1776931158160.png)

* 高频头上标有对应标识

![1775028326336](image/准备通信链路/1775028326336.png)

* 对应拧紧天线，确保连接稳定

![1776930853097](image/准备通信链路/1776930853097.png)

* 另一侧是NANO仓快拆口，用于连接遥控器

![1776930989508](image/准备通信链路/1776930989508.png)

3. 通过type-c转type-c数据线将高频头与手机连接

![1775028603242](image/准备通信链路/1775028603242.png)

* 如图，使用TypeC To TypeC数据线连接高频头与手机即可

![1776930167243](image/准备通信链路/1776930167243.png)

* 可使用手机支架固定手机和遥控器，确保连接稳定

![1776929766117](image/准备通信链路/1776929766117.png)

![1776929793599](image/准备通信链路/1776929793599.png)

## 软件准备

### 有线连接

**手机连接**

* 手机与高频头连接后，打开手机安卓版QGC软件，点击左上角图标，进入设置界面，下滑找到“通讯连接”选项，点击添加连接，连接类型选择“串口”，波特率改为“115200”

![1769508691645](image/准备通信链路/1769508691645.png)

![1769508777312](image/准备通信链路/1769508777312.png)

![1775029383524](image/准备通信链路/1775029383524.png)

* 添加好名称后，点击保存，再点击连接，即可自动连接上地面站

![1775029501552](image/准备通信链路/1775029501552.png)

**电脑连接**

+ 若想连接电脑使用QGC软件，需要先下载[CH340驱动](https://espjs.cn/driver)

![1775030599071](image/准备通信链路/1775030599071.png)

* 电脑与高频头连接后，打开电脑版QGC软件，点击左上角图标，进入设置界面，下滑找到“通讯连接”选项，点击添加连接，连接类型选择“串口”，波特率改为“115200”，串口选用连接到高频头后出现的串口，如图：com20

![1775033535607](image/准备通信链路/1775033535607.png)
